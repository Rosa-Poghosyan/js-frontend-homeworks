# Shopping Card 🛒

Welcome to the **Shopping Card** project! This is a simple and interactive shopping cart application built as part of a JavaScript frontend homework assignment. It allows users to browse products, add them to a cart, adjust quantities, and view the total price, all within a responsive and user-friendly interface.

---

## ✨ Overview

The **Shopping Card** application demonstrates core frontend development skills using JavaScript, HTML, and CSS. It provides a seamless shopping experience with dynamic cart updates and responsive design, making it accessible across devices.

---

## 🛠️ Features

- **Product Listing**: Browse a visually appealing list of products with names, prices, and images.
- **Add to Cart**: Easily add products to your cart with a single click.
- **Cart Management**: Adjust item quantities or remove items from the cart.
- **Total Price Calculation**: Automatically updates the total price as you modify the cart.
- **Responsive Design**: Optimized for both desktop and mobile devices for a consistent experience.

---

## 💻 Technologies Used

- **HTML**: Provides the structure of the web application.
- **CSS**: Ensures a clean, modern, and responsive layout.
- **JavaScript**: Powers the dynamic cart functionality and real-time updates.

## 🚀 Getting Started

### Prerequisites
- A modern web browser (e.g., Chrome, Firefox, Safari).
- **Node.js** and **npm** installed on your machine.
- Basic understanding of HTML, CSS, and JavaScript.

### Installation
1. **Clone the repository**:
   ```bash
   git clone https://github.com/Rosa-Poghosyan/js-frontend-homeworks.git
   ```
2. **Navigate to the project folder**:
   ```bash
   cd js-frontend-homeworks/shopping-card
   ```
3. **Install dependencies**:
   ```bash
   npm install
   ```
4. **Start the development server**:
   ```bash
   npm run dev
   ```
5. **Open the application**: Navigate to the address provided in the terminal (e.g., `http://localhost:3000`) in your browser.

---

## 📖 Usage

1. **Browse Products**: Explore the product list on the homepage.
2. **Add to Cart**: Click "Add to Cart" to include a product in your shopping cart.
3. **Manage Cart**: View your cart, adjust item quantities, or remove items as needed.
4. **View Total**: The total price updates automatically with every change.
